Accessibility

Practical accessibility commitments

The site is built with semantic HTML, strong contrast, keyboard support, clear labels, and plain language.

Current support

  • Skip link to main content
  • Visible labels for every calculator input
  • ARIA live regions for calculator status updates
  • Visible focus states
  • Responsive cards and layouts
  • Reduced-motion respect where supported

Design approach

The site avoids flashing animation, avoids color-only status communication, and is intended to remain usable at 200% zoom.

Need a correction?

If you find an accessibility issue, use the contact details on the contact page.
